# Noise

Noise is an iOS and Android blogging app for couples to keep their everyday memories safe and vivid forever. They can find any memory easily at any time.

# Source code: 
Hidden.

# Demo:
https://youtu.be/STAyoGhPgKM?si=s-s8_kBsDfIonrE8

# Tech-stack:
- M: MongoDB
- E: Express.js
- R: React Native
- N: Node.js
